# ValueSim

This is a collection of tools to check and compare different potential investments and
their (long time) revenue.

### What do we aim for

#### Expose total value over time

* Liquid Value
* Fortune
* Debt
* Passive Income

#### Track of different types of buckets

* Current Account
    * Value
* Loan
    * Rate
    * Interest
* Real Estate
    * Apreciation
    * Rent
    * Taxes
    * Maintenance
* Stocks
    * Apreciation
    * Dividends
* Economy Deposits
    * Interest
    * Duration

Actions:

* Save
* Buy
* Sell
* Exit
